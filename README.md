# Study Buddy App

## Project Overview

Study Buddy is a peer-to-peer study support web application designed to connect university students who want to collaborate academically. 

The platform enables students to:
- Create user profiles
- Offer help in subjects they are confident in
- Request support in subjects they find challenging
- Browse available study partners

This project aligns with the 2026 theme:
**"Sharing, exchange and building community"**
by promoting knowledge exchange without financial gain.
